# Stroke Risk Classification Using Machine Learning

## Overview

This project compares four supervised machine learning algorithms for predicting stroke risk using patient demographic and clinical information.

The project evaluates:

- Logistic Regression
- Gaussian Naïve Bayes
- Support Vector Machine (SVM)
- K-Nearest Neighbours (KNN)

The goal is to identify the most suitable model for healthcare screening by evaluating performance using Accuracy, Precision, Recall, F1-score, and ROC-AUC.
